## To Build Mac Version

### You Need a Mac Computer

**If you have a Mac:**

1. Open Terminal
2. Run these commands:
   ```bash
   cd ~/Documents
   git clone https://github.com/Adela-Cal/PDF-to-Mail-1.0.git
   cd PDF-to-Mail-1.0
   chmod +x SETUP_AND_BUILD_MAC.sh
   ./SETUP_AND_BUILD_MAC.sh
   ```
3. Wait 20 minutes
4. Get: `Speedy Statements-1.0.0.dmg`

**If you DON'T have a Mac:**
