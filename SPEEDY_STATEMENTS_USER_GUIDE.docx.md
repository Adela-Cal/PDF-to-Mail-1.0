## 1. Introduction

**Speedy Statements** is a desktop application designed to streamline your workflow by:

✓ Extracting email addresses from PDF files  
✓ Managing reusable email templates  
✓ Creating Outlook-ready draft emails with PDF attachments  
✓ Saving time on repetitive email tasks  

This guide will walk you through every step of the process, from preparing your PDFs to sending draft emails.

## 8. Troubleshooting

### Issue: PDFs Not Loading

**Symptom**: No PDFs appear after selecting a folder

**Solutions**:

1. Verify folder contains PDF files (not Word docs or other formats)
2. Check PDFs are not password-protected
3. Ensure PDFs contain actual text (not scanned images)
4. Try selecting the folder again

---

### Issue: No Email Addresses Extracted

**Symptom**: PDFs load but email column is empty

**Solutions**:

1. Open PDF manually and verify it contains email addresses
2. Check emails are in standard format (name@domain.com)
3. PDFs with emails as images cannot be extracted
4. Try a different PDF to test

---

### Issue: Outlook Drafts Won't Open

**Symptom**: Double-clicking .eml file does nothing or shows error

**Solutions**:

1. Verify Microsoft Outlook is installed
2. Set Outlook as default email program:
   - Settings → Apps → Default apps → Email → Select Outlook
3. Right-click .eml file → "Open with" → Choose Outlook
4. Restart computer and try again

---

### Issue: Draft Opens as Read-Only

**Symptom**: Cannot edit the draft in Outlook

**Solutions**:

- This should not happen in version 1.0.0
- If it occurs: Close Outlook, delete .eml file, generate new draft
- If problem persists: Reinstall Speedy Statements

---

### Issue: PDF Not Attached to Email

**Symptom**: Draft opens but PDF is missing

**Solutions**:

1. Verify PDF file still exists in original location
2. Try generating draft again
3. Manually attach PDF in Outlook if needed

## 9. Tips & Best Practices

### Organizing Your PDFs

- Keep all statement PDFs in a dedicated folder
- Use clear, consistent naming for PDF files
- Archive old PDFs after processing

### Using Templates Effectively

- Create separate templates for different purposes
- Use descriptive template names
- Update templates regularly to keep content fresh

### Managing Multiple Accounts

- Add all your company email accounts
- Use descriptive names to distinguish accounts
- Select appropriate account for each batch

### Workflow Efficiency

1. Export all PDFs first
2. Set up templates in advance
3. Process PDFs in batches
4. Review one draft before generating all
5. Send emails in groups from Outlook

## 10. Additional Information

### Data Storage

All templates and accounts are stored locally at:

```
C:\Users\[YourName]\AppData\Roaming\SpeedyStatements\data\
```

This data is specific to your installation and not shared between computers.

### System Requirements

- Windows 10 or Windows 11
- Microsoft Outlook (for opening .eml draft files)
- 200 MB free disk space
